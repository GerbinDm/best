    1. Global attributes

      Глобальные атрибуты могут быть указаны для любых элементов HTML, даже для тех, которые не указаны в стандарте. Атрибутами являются те же , class и ID к примеру, которые вы можете дать любому элементу, чтобы применить к ним впоследствии в CSS разные свойства, а именно задать цвета, отступы и прочее. Так же глобальным атрибутом является style(это применение стилей прямо в HTML документе, что очень не рекомендуется, кстати. Данный атрибут будет имень очень высокий приоритет(смотрите приоритеты стилей в selectors) и в последствии может нарушить целостность вашего дизайна на странице, поэтому лучше им пользоватся в крайних случаях, когда вам нужно перекрыть новым стилем какой-то конкретный элемент в частности. С другими атрибутами вы можете ознакомится во вкладке Global attributes.

    2. Events

      События HTML - это специальные глобальные атрибуты, используемые в тегах для вызова обработчиков событий, написанных на различных языках сценариев таких, как JavaScript и вызываемых, когда на странице происходит какое-либо действие. События позволяют сделать вашу страницу динамической. Хорошим примером является onclick ивент, который позволяет по клику вызывать функцию из файла JS. Строчкой кода в HTML подключается файл Js, далее к нужному вам элементу(кнопке к примеру) прописывается onclick и функция которую вам нужно по клику вызывать из файла. В результате когда вы нажмете на кнопку выполнится функция, которую вы задали.
