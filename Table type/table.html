    Теги делятся на табличные, блочные и строчные. Табличные и блочные теги могут содержат в себе строчные. Каждая HTML-страница состоит из набора тегов , которые вы можете назвать строительными блоками веб-страниц. Они создают иерархию, которая структурирует контент по разделам, параграфам, заголовкам и другим блокам. Тег может быть одинарным, может быть парным.(разница в синтаксисе) Слева можете ознакомится с каждым типом в частности. Нажмите Inspect и вы увидете набор тегов разметки данной страницы.
    Очень общая схема работы с тегами в HTML документе:
    1. Задали Tag
    2. Присвоили Class
    3. Выбрали класс селектором и задали ему атрибут. Классы смотрим во вкладке Classes в топ меню
    Для лучшего освоения материала советую также параллельно смотреть в W3Schools, который содержит более полную информацию по каждому из тегов и атрибутов(нажми на меня)
