      Pseudo elements
    1. General

      Знак :: позволяет выбрать вещи, которых нет в HTML.

      Пример: p::first-line соответствует первой линии абзаца p.

    2. ::after

      В CSS, ::after создаёт псевдоэлемент, который является последним потомком выбранного элемента. Часто используется для добавления косметического содержимого в элемент с помощью свойства content. По умолчанию является инлайновым.

      Задача:/* Добавить стрелки после ссылок */

      a::after {

      content: "→"; }

    3. ::before

      В CSS, ::before создаёт псевдоэлемент,который является первым потомком выбранного элемента. Часто используется для добавления косметического содержимого в элемент с помощью свойства content. По умолчания является инлайновым.

      Задача:/* Добавить сердце перед ссылками */

      a::before {

      content: "♥"; }

    4. ::first-letter

      CSS псевдоэлемент ::first-letter применяет стили к первой букве первой строки блочного элемента, но только если нету другого предшествующего содержимого (такого как изображения или инлайн таблицы).

      Задача:/* Стили для первой буквы элемента "p" */

      p::first-letter {

      font-size: 130%;}

    5. ::selection

      Псевдоэлемент ::selection позволяет применить стили к части документа, который был выделен пользователем (например, с помощью мыши).

      Задача:/* применить стили к части документа, который был выделен пользователем */

      ::selection {

      background: cyan; }

    6. ::backdrop

      CSS псевдо-элемент ::backdrop это прямоугольник с размерами окна, который отрисовывается сразу же после отрисовки любого элемента в полноэкранном режиме. Это включает элементы, установленные в полноэкранный режим с помощью Fullscreen API и элементы "dialog".

      Задача:/* Подложка показывается только тогда, когда диалоговое окно открыто с помощью метода dialog.showModal() */

      dialog::backdrop {

      background: rgba(255,0,0,.25);

      }
