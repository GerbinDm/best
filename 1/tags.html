    Для того, чтобы создать страницу, вам нужно разбить страницу на блоки, строки и таблицы, которые вы будете заполнять контентом. Для этого вам помогут теги соответсвенных типов. Их следует располагать в так называемый "каркас" состоящий из HTML и BODY. За стили, в которые входят пропорции, цвета, шрифты и прочее ваших блоков, строк и таблиц отвечают атрибуты(см.вкладку выше). Прописываются атрибуты в CSS.
    Как создать страницу:
    1. Задали Tag в HTML документе
    2. Присвоили Class в HTML документе
    3. Выбрали атрибут в CSS при помощи селекторов. Селекторы смотрим во вкладке выше
    Для лучшего освоения материала советую также параллельно смотреть в W3Schools, который содержит более полную информацию по каждому из тегов и атрибутов(нажми на меня)
