      General info
    1. Global attributes

      Глобальные атрибуты могут быть указаны для любых элементов HTML, даже для тех, которые не указаны в стандарте. Это значит, что все нестандартные элементы должны допускать эти атрибуты, даже если использование этих элементов означает, что документ больше не является HTML5 -совместимым. Например, HTML5 -совместимые бразузеры скрывают содержимое, помеченное как < foo hidden >...< foo >, хотя < foo > не является верным HTML элементом.

    2. Events

      События HTML - это специальные глобальные атрибуты, используемые в тегах для вызова обработчиков событий, написанных на различных языках сценариев таких, как JavaScript и вызываемых, когда на странице происходит какое-либо действие. События позволяют сделать вашу страницу динамической.

    Важно помнить:
    1. Данные выше теги - это не полный их список
    2. Теги крайне произвольные сами по себе. Вы можете использовать для контента как main так и div, так и section к примеру
    3. У них, конечно, разные свойства, но для выполнения вашей задачи зачастую подходит множество тегов. Вы можете делать как угодно. Но помните, что код должен быть как можно более логичен и прост. В этом помогут общие стандарты, которые хорошо описаны во многих гайдах для новичков.
