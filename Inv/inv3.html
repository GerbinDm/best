    1. Каждая HTML-страница состоит из набора тегов , которые вы можете назвать строительными блоками веб-страниц.

    2. Они создают иерархию, которая структурирует контент по разделам, параграфам, заголовкам и другим блокам.

    3. Теги - это основные конструкции (команды) языка HTML.
    4. Благодаря тегам любой браузер понимает, каким смыслом вы наделяете ту или иную часть веб-страницы.
    Очень общая схема работы с тегами в HTML документе:
    1. Задали Tag
    2. Присвоили Class
    3. Выбрали класс селектором и задали ему атрибут. Классы смотрим во вкладке Classes в топ меню
